# Personal Portfolio Website — ASP.NET

A multi-page personal portfolio web application built with ASP.NET Web Forms, showcasing my background, skills, projects, and a fully functional user authentication system.

---

## 📄 Pages

| Page | Description |
|---|---|
| Home | Introduction with profile photo and bio |
| About | Background, skills, and academic focus |
| Projects | Showcase of academic and personal projects |
| Contact | Contact form with Name, Email, and Message |
| Login | User login with email and password |
| Register | New user account registration form |

---

## ✨ Features

- Multi-page navigation (Home · About · Projects · Contact · Login · Register)
- User authentication — Register and Login functionality
- Profile photo display on homepage
- Contact form with Send Message functionality
- Consistent header and footer across all pages
- Green-themed responsive design

## 🚀 Getting Started

1. Clone the repository
2. Open the solution in **Visual Studio**
3. Update the database connection string in `Web.config`
4. Run database migrations or execute the SQL setup script
5. Press **F5** to run locally on `https://localhost:44332`
